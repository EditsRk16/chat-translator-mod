
# Chat Translator Mod

[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.21%2B-62b47a)](https://minecraft.net)
[![Fabric](https://img.shields.io/badge/Fabric-1.21+-purple)](https://fabricmc.net/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Um mod leve e eficiente para traduzir mensagens do chat do Minecraft em tempo real.

---

## ✨ Funcionalidades

- **Tradução em tempo real** do chat
- Suporte a **múltiplos idiomas** (detecção automática + escolha manual)
- Tradução seletiva (apenas mensagens em outro idioma)
- Tradução de mensagens do servidor, jogadores e sistema
- Interface simples com tecla de atalho (**K**)
- Suporte a **Forge** e **Fabric**
- Totalmente configurável
- Baixo impacto no desempenho

## 🛠 Idiomas Suportados

- Português (Brasil e Portugal)
- Inglês
- Espanhol
- Francês
- Alemão
- Italiano
- Russo
- Japonês
- Coreano
- Chinês (Simplificado e Tradicional)
- E muitos outros

## 📥 Instalação

1. Baixe a versão mais recente na aba **[Releases](https://github.com/EditsRk16/chat-translator-mod/releases)**
2. Coloque o arquivo `.jar` na pasta `mods`
3. Inicie o Minecraft (Fabric)
4. Pressione **K** para abrir o menu de configuração

## ⚙️ Configuração

Pressione **K** para abrir o menu gráfico de configurações.

Arquivo de configuração também disponível em:
```
config/chat-translator/config.toml
```

### Principais opções:
- Idioma alvo
- Ativar/desativar tradução automática
- Provedor de tradução
- Tecla de atalho (padrão: **K**)
- Prefixo nas mensagens traduzidas


## 🚀 Como Usar

1. Instale o mod
2. Entre em um servidor
3. Pressione **K** para configurar seu idioma
4. As mensagens em outros idiomas serão traduzidas automaticamente
